### Prakriti Assessment Score

```python
def calculate_prakriti(vata_score: int, pitta_score: int, kapha_score: int) -> str:
    """
    Determine Prakriti (constitution) from questionnaire scores
    
    > "‡§™‡•ç‡§∞‡§ï‡•É‡§§‡§ø‡§Ç ‡§Ø‡§æ‡§§‡§ø ‡§≠‡•Ç‡§§‡§æ‡§®‡§æ‡§Ç..."

    > ‚Äî Bhagavad Gita 3.33
    """
    total = vata_score + pitta_score + kapha_score
    v_pct = vata_score / total * 100
    p_pct = pitta_score / total * 100
    k_pct = kapha_score / total * 100
    
    # Determine dominant Dosha(s)
    if v_pct > 50:
        return "Vata Prakriti"
    elif p_pct > 50:
        return "Pitta Prakriti"
    elif k_pct > 50:
        return "Kapha Prakriti"
    elif v_pct > 33 and p_pct > 33:
        return "Vata-Pitta Prakriti"
    elif p_pct > 33 and k_pct > 33:
        return "Pitta-Kapha Prakriti"
    elif v_pct > 33 and k_pct > 33:
        return "Vata-Kapha Prakriti"
    else:
        return "Tridoshic (Sama) Prakriti"  # Rare!
```
