#### Layer 6: Presentation — Mantra (मन्त्र)

```
FUNCTION: Properly formatted commands

MANTRA STRUCTURE:
=================

[HEADER] + [ADDRESS] + [COMMAND] + [TERMINATOR]

HEADER = OM (ॐ)
• Boot sequence
• Authentication
• "I am a valid node"

ADDRESS = Deity Name or Bija
• Who/what you're connecting to
• Examples: Ganapataye, Shivaya, HREEM

COMMAND = Shakti words
• What you want to communicate
• Examples: Namah, Svaha, Vidmahe

TERMINATOR = Execution command
• How to execute
• NAMAH = Surrender/Receive
• SVAHA = Offer/Transmit
• PHAT = Destroy/Clear

EXAMPLE:
========
ॐ नमः शिवाय
OM NAMAH SHIVAYA

OM = Header (I am valid node)
NAMAH = Command (I bow/surrender)
SHIVAYA = Address (to Shiva)

Complete meaning: "I authenticate and surrender to Shiva"
```

### 4.2 Debugging Protocol

```
IF mantra not working:
    1. CHECK pronunciation
       → Listen to authentic source
       → Practice with teacher
       
    2. CHECK intention
       → Is it aligned with Dharma?
       → Is it clear and specific?
       
    3. CHECK consistency
       → Are you practicing regularly?
       → Same time, same place?
       
    4. CHECK purity
       → Physical cleanliness?
       → Mental clarity?
       
    5. CHECK initiation
       → Was mantra properly received?
       → From qualified source?
       
    6. CHECK patience
       → Have you given it enough time?
       → Some mantras need years
       
    7. CHECK karma
       → Is this result meant for you?
       → What is your prarabdha?
```
